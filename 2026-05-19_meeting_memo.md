## Suggested Project Ideas

###  🔄 Импорт траекторий кодинг-агентов
- Пайплайн по импорту траекторий из Cursor, Claude Code, Codeux и прочих
- **Важно:** чтобы всё чётко работало и было ясно, где subagent, где compaction и т.п.

**​Плюсы:**
- High-value проект.
- Норм параллелится:
    - ведущие инженеры проектируют общую архитектуру ядра парсера и схему данных
    -  инженеры реализации пишут конкретные адаптеры под каждый инструмент.
      
**​Минусы:**
- слабая стандартизация логов закрытых инструментов
- рутина с JSON

​Стек: Python (Pydantic для валидации схем, Click/Typer для CLI), JSON/YAML-парсеры.

### Доступы и Риски (РФ):
- Низкие. Доступ к API моделей не нужен — работа с готовыми файлами логов.
- Единственная сложность — выкачать сами файлы логов/траекторий из-за платных или заблокированных аккаунтов Cursor/Claude Code.
  
### Ресурсы
- Оценка сложности: 6/10
- ​Итоговая оценка: 8.5 / 10
- ​Оценка времени: ~160–200 часов.
- ​Распределение:
      - ~40 ч. (Ведущие) + ~140 ч. (Прикладные)

### 🛠️ Улучшение тул-коллинга
- Поддержать **Codemode (Cloudflare)** или аналогичную штуку от Anthropic
- Разработать более эффективный tool calling

**Плюсы:**
- Работа со свежими протоколами.
- Повышает автономность агентов в эвалах.
  
**​Минусы:**

- Anthropic жесточайше банит за малейшие намёки на обход блокировок из РФ.
  
### Ресурсы
